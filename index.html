WattDecide is for decision support and education – not financial advice.

Why this is different

  • ✓ Independent by default – the model doesn’t “want” you to buy anything.
  • ✓ No hidden defaults – assumptions are visible and editable (including the uncomfortable ones).
  • ✓ Anti-greenwashing – results are conditional: “EV wins if…” / “solar wins if…”.
  • ✓ Audit trail – every headline number can be traced back to a specific input.

Antifragile method

  • ✓ Stress-case scenarios are first-class (not an afterthought).
  • ✓ Sensitivity shows which 2–3 assumptions actually drive the result.
  • ✓ “Break conditions” – what must change for the conclusion to flip.
  • ✓ Versioned changes with short release notes (no silent tweaks).

In v1

  • ✓ Total cost and cashflows with an explicit time horizon and inputs.
  • ✓ Break-even + NPV, plus “what would need to be true” sensitivity.
  • ✓ Baseline + stress-case scenarios (kept small and realistic).
  • ✓ Output you can save/share with assumptions attached.
Scope is intentional: the goal is a decision you can trust, not a black-box optimiser. Localisation (tariffs, incentives, taxes) can come later as optional packs if demand forces it.

What you get

  • ✓ A clean inputs sheet (no hunting across tabs).
  • ✓ Clear outputs: NPV, cost per km / per kWh, and break-even timing.
  • ✓ A sensitivity view that answers: “what matters, what doesn’t, and why”.
  • ✓ A short readme explaining interpretation and limitations.
